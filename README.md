# Hyper-CMDB

## 技术栈

### 后端技术栈

- 语言：Go
- 框架：gin
- 数据库：MySQL Redis
- ORM：gorm
- 消息队列：asynq
- 日志：logrus
- 认证：jwt
- 权限控制：casbin
- 配置管理：viper

### 前端技术栈

- 语言：JavaScript
- 框架：React
- UI组件库：Ant Design
- 构建工具：Vite
- HTTP客户端：Axios

## 多云支持

- 阿里云
- Azure
- AWS
- 京东云

## 监控

- 监控：prometheus+grafana
- 日志：ELK
